---
title: "Styling"
---

Styling is essential part of every web application, so we tried to make it as fact and convenient as possible, with the help of [Mantine UI](https://v6.mantine.dev/) library. It provides huge variety of customizable components and hooks.

Mantine's [theme object](https://v6.mantine.dev/theming/theme-object/) makes it possible to build your own design system on top of library.
